Microservice description

 

To date, there are several custom computer configuration and component benchmarking websites. The data on which these services rely are not always updated in real-time, which undermines the relevance of their recommendations to users. Our API will address the same user needs in a entralized manner and through a simplified user interface, while ensuring the accuracy and comprehensiveness of the data provided to the user (technical compatibility, current prices, etc.).
